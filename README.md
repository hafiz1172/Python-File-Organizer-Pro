# 📂 Professional File Organizer (PyQt5)

A desktop application that automatically organizes files into categorized folders based on their extensions.

### ✨ Features:
- **GUI Interface:** Easy to use with "Browse" and "Organize" buttons.
- **Grouping:** Categorizes files into Images, Documents, Videos, etc.
- **Safety:** Handles duplicates and avoids crashing during file locks.
- **Logging:** Keeps a record of moved files.

### 🛠️ How to Run:
1. Install requirements: `pip install PyQt5`
2. Run the script: `python main.py`
